# Post-push Verification
| **Command**                                   | **Description**                                                                                 |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------|
| **Post-Push Verification -  Common Commands** |                                                                                                 |
| `git log --oneline --graph --all`             | üìä Visualize commit history across all branches                                                 |
| `git diff origin/main`                        | üîç Compare local branch state against remote main                                               |
| `git log main..origin/main`                   | üìå Show commits in remote main not yet in local main                                            |
| `git log origin/main..main`                   | üìå Show commits in local main not yet in remote main                                            |
|                                               |                                                                                                 |
| **List Commits in Remote Branch**             |                                                                                                 |
| `git log origin/main --oneline -5` + enter    | üîç View the history that has been pushed and is currently on the remote server                  |
|                                               |                                                                                                 |
| **Check if a Specific Commit is Pushed**      |                                                                                                 |
| `git log @{push} -1` + enter                  | üîç View if a particular local branch has been synchronized with its remote branch (@{push})     | 
| `git branch -v` + enter                       | üîç View local branches and show if they are "ahead" or "behind" their remote trackers.          |
| `git branch -vv` + enter                      | üîç View local branches with detailed info about their relationship to remote-tracking branches. |
| `git branch -a -v` + enter                    | üîç View both local and remote-tracking branches with their latest commit details.               |
|                                               |                                                                                                 |
