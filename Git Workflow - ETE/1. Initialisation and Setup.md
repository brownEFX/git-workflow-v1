# Basic git Workflow for a new project
| ***Stage***                                                   | ***Key Commands***                                                                           |
|---------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| ***Phase 0: Setup Working Directory***                        | WD for a new project                                                                         |
| `cd \` then `mkdir my-new-project` + enter                    | Go to root dir [C:\], Create project Working Directory                                       | 
| `cd my-new-project` + enter                                   | Navigate to project Working Directory                                                        |
| `echo "" > README.md` enter                                   | Add initial project files                                                                    | 
| `ls` enter                                                    | View contents of current dir                                                                 |
|                                                               |                                                                                              |
| Initialization: `git init -b main` + enter                    | Initialise Git repo with main branch                                                         | 
| Verify: `git status` or `git status -s` + enter               | Verify current state                                                                         | 
| Staging: `git add <file-name>` OR `git add .` + enter         | Add file(s) to staging area                                                                  | 
| Verify: `git ls-files -c/-o/-m/-d` + enter                    | View tracked files in repo                                                                   |
|                                                               |                                                                                              | 
| `echo "" > .gitignore` enter                                  | Add and update .gitignore file patterns for files to ignore                                  |
| Staging: `git add .gitignore` + enter                         | Add file(s) to staging area                                                                  |
| `git rm --cached <file-path>` + enter                         | Unstage file from staging area and remove from index, keep it in WD                          |
| Verify: `git ls-files -c/-o/-m/-d` + enter                    | View tracked files in repo                                                                   |
|                                                               |                                                                                              | 
| Commit: `git commit -m "commit message"` + enter              | Commit changes with message                                                                  |
| `git log` + enter                                             | View commit history                                                                          |  
| `git log --graph --oneline --decorate -5`  + enter            | View commit history as a gra.                                                                |  
| `git log --stat --since="2 days ago"` + enter                 | View commit history with file changes summary for the last 2 months.                         |
| `git log -p <commitId>` + enter                               | View changes made in a specific commit.                                                      | 
|                                                               |                                                                                              |  
| Connect to Remote: `git remote add origin + repo URL` + enter | üåê Add remote repo                                                                           |
| Verify: `git remote -v` + enter                               | ‚úÖ View remote repo                                                                           |
| Verify: `git remote show origin` + enter                      | ‚úÖ View remote repo details                                                                   |
|                                                               |                                                                                              | 
| Initial Push to Remote: `git push -u origin main` + enter     | Push current branch to remote repo and set upstream                                          | 
| Verify: `git ls-tree -r origin/main --name-only` + enter      | üóÉÔ∏è List files in branch                                                                     |
| Verify: `git log --oneline --graph --all` + enter             | View graphical view of all commits across branches.                                          |
| Verify: `git log origin/main` + enter                         | View the commit history of the remote branch main.                                           |
|                                                               |                                                                                              |
| Verify: `git fetch origin` + enter                            | Update remote-tracking branches                                                              | 
| Verify: `git status -v` + enter                               | View files are changed, staged, or untracked + detailed preview of the changes               | 
| Verify: `git log --oneline --graph --all` + enter             | View graphical view of all commits across branches.                                          |
| Verify: `git log origin/main` + enter                         | View the commit history of the remote branch main.                                           |
| Verify: `git log main..origin/main` + enter                   | Compare remote with local branch                                                             |
| Verify: `git fetch` then `git diff origin/main` + enter       | View differences between your local working tree and the remote .                            | 
| Verify: `git branch -vv` + enter                              | View local branches with detailed info about their relationship to remote-tracking branches. |
