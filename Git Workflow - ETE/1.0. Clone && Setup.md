# Basic git Workflow for a new project
| ***Stage***                                                                | ***Key Commands***                                                                           |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| ***Step 0: Setup Working Directory***                                      | WD for a new project                                                                         |
| `cd \` then `mkdir myWorkspace` + enter                                    | Go to root dir [C:\], Create project Working Directory                                       | 
| `cd myWorkspace` + enter                                                   | Navigate to project Working Directory                                                        |
| `ls` enter                                                                 | View contents of current dir                                                                 |
|                                                                            |                                                                                              |
| ***Step 1: Clone Repo***                                                   | Cloning creates a local copy of a remote repository                                          |
| `git clone <repository_url>` + enter                                       | Create a local copy (clone) of a remote Git repository.                                      | 
| `git clone <repository_url> <local_directory_name>` + enter                | Clone the repository into a specific local folder,                                           | 
| `git clone -b <branch_name> <repository_url>` + enter                      | Clone only a single branch instead of the default branch.                                    | 
| `git clone --depth 1 <repository_url>` + enter                             | Limit how much commit history is downloaded to save time and space.                          |
| `git clone git@github.com:user/repository.git` + enter                     | If you have SSH keys set up, use the SSH URL for password-less authentication.               |
|                                                                            |                                                                                              | 
| `cd <repository_name>` + enter                                             | Navigate to local repo.                                                                      |
| Verify: `git status` + enter                                               | Verify branch and update status                                                              |
| Verify: `git ls-files -c/-o/-m/-d` + enter                                 | View tracked files in repo                                                                   |
| Verify: `git remote -v` + enter                                            | ‚úÖ View remote repo                                                                           |
| Verify: `git remote show origin` + enter                                   | ‚úÖ View remote repo details                                                                   |
|                                                                            |                                                                                              |
| ***Step 2: Create and Switch to a New Feature Branch***                    |                                                                                              |
| `git pull origin main` + enter                                             | Sync main                                                                                    |
| `git switch -c <feature/branch_name>` + enter                              | Create and switch to new branch.                                                             | 
| Verify: `git status` + enter                                               | Verify branch and update status                                                              |
| Verify: `git branch --show-current` + enter                                | Verify current branch                                                                        |
| Verify: `git branch` + enter                                               | List local branches                                                                          |
| `git branch -a` + enter                                                    | üîç View all branches incl. local and remote-tracking branches                                |
| `git remote show <remote-name>` + enter                                    | üîç View detailed information about a specific remote                                         |
|                                                                            |                                                                                              |
| ***Step 3.1.: Make Changes and Initial Commit***                           |                                                                                              |
| Add/Modify Source Code and Project Files                                   | Add new and/or modify existing code                                                          |
| `git add .` or `git add <file>` + enter                                    | Add files to Staging Area                                                                    |
| `git commit -m "commit message"` + enter                                   | Bundle those changes into a commit                                                           |
|                                                                            |                                                                                              |
| Verify: `git status` + enter                                               | Verify branch and update status                                                              |
| Verify: `git log` + enter                                                  | Confirm that new commit appears at the top of the history                                    |
| `git log --graph --oneline --decorate -5`  + enter                         | View commit history as a gra.                                                                |  
| `git log --stat --since="2 days ago"` + enter                              | View commit history with file changes summary for the last 2 months.                         |
| `git log -p <commitId>` + enter                                            | View changes made in a specific commit.                                                      | 
|                                                                            |                                                                                              |
| ***Step 3.2.: Make Further Changes and Commit***                           |                                                                                              |
| Add new Source Code                                                        | Add `feature-page.java` and `feature-tests.java`                                             |
| Modify existing Source Code and Project Files                              | Update `ete-tests.java`                                                                      |
|                                                                            |                                                                                              |
| `git add .` or `git add <file>` + enter                                    | Add files to Staging Area                                                                    |
| `git commit -m "commit message"` + enter                                   | Bundle those changes into a commit                                                           |
| Verify: `git cherry -v` + enter                                            | üîç View list of commits waiting to be pushed, marked with a + sign.                          |
| Verify: `git log '@{u}..'` + enter                                         | üîç View what is local and not yet pushed                                                     |
| To Push: `git rev-list --count @{push}..HEAD` + enter                      | üîç View number of commits that are not in remote                                             |
|                                                                            |                                                                                              |
| ***Step 3: Push to remote***                                               |                                                                                              |  
| Initial Push to Remote: `git push -u origin <feature/branch-name>` + enter | Push current branch to remote repo and set upstream                                          | 
| Verify: `git log @{push} -1` + enter                                       | üîç View if a particular local branch has been synchronized with its remote branch (@{push})  |
|                                                                            |                                                                                              |
| ***Step 4: PR && Merge***                                                  |                                                                                              |
| GitHubUI: Pull Request Process                                             | üñ• Open PR ‚Üí Review ‚Üí Approve ‚Üí Merge                                                        | 
|                                                                            |                                                                                              |
| **Post-Merge Verification**                                                |                                                                                              |
| Verify: `git log --oneline --graph --all`                                  | üìä Visualize commit history across all branches                                              |
| Verify: `git diff origin/main`                                             | üîç Compare local branch state against remote main                                            | 
| Verify: `git log main..origin/main`                                        | üìå Show commits in remote main not yet in local main                                         | 
| Verify: `git log origin/main..main`                                        | üìå Show commits in local main not yet in remote main                                         | 
| Verify: `git diff '@{u}'` + enter                                          | üîç View differences between your local code and a remote.                                    |
| Verify: `git log origin/main..main --oneline`  + enter                     | üîç View commits in local main not yet in remote main Use after Sync                          |
|                                                                            |                                                                                              |
| Verify: `git status -v` + enter                                            | View files are changed, staged, or untracked + detailed preview of the changes               |
| Verify: `git branch -vv` + enter                                           | View local branches with detailed info about their relationship to remote-tracking branches. |
| Verify: `git log --merges --oneline --graph`                               |                                                                                              |
|                                                                            |                                                                                              |
| Verify: `git branch --merged` + enter                                      | View branches have been merged into your current branch.                                     |
| Verify: `git log main..<feature/feature-name>` + enter                     | View commits in feature/feature-name but not in main.                                        |
| Verify: `git diff origin/main` + enter                                     | View differences between your local branch and the remote main.                              |
| Verify: `git log --merges --oneline --graph` + enter                       | View all merge commits with full details (commit hash, author, date, message).               |
| Verify: `git log --merges -m --name-only -1` + enter                       | View files modified in the most recent merge commit.                                         |
| Verify: `git log main..<feature/feature-name>` + enter                     | View commits in feature/feature-name but not in main.                                        |
| Verify: `git log  <feature/feature-name>` + enter                          | View the full commit history of the specified feature branch.                                |
| Verify: `git diff '@{u}'` + enter                                          | View differences between your local code and a remote.                                       |
|                                                                            |                                                                                              | 
| **Step 5: Sync Local after Merge**                                         |                                                                                              |
| Branch: `git switch main` + enter                                          | üìå Switch to main                                                                            |
| Pre-Pull: `git rev-list --count HEAD..@{push}` + enter                     | üîç Shows how many commits the remote has that you do not                                     |
|                                                                            |                                                                                              |
| **Fetch, Review, Merge**                                                   |                                                                                              |
| Get Latest: `git fetch origin` + enter                                     | üìå Download latest from remote                                                               |
| Verify: `git diff origin/main` + enter                                     | üìå View differences between your local working tree and the remote.                          | 
| Verify: `git diff origin/feature..feature` + enter                         | üìå View what is in feature that is not in origin/feature                                     |
| Review: `git log origin/main`                                              | üìå View commit history of remote main                                                        |
| Verify: `git log main..origin/main` + enter                                | üìå Compare remote with local branch                                                          |  
|                                                                            |                                                                                              |
| Merge: `git merge origin/main`                                             | üìå Integrate changes from remote main                                                        |
| Verify: `git status`                                                       | üìå Check if local is ahead/behind                                                            |
| Verify: `git log --oneline --graph --decorate --all`                       | üìå Visualize commit graph relative to remote main                                            |
|                                                                            |                                                                                              |
| **Quick Sync**                                                             |                                                                                              |
| `git pull origin main` + enter                                             | üîÑ Fetch && Merge                                                                            |
|                                                                            |                                                                                              |
| **Step 5: Sync Local after Merge**                                         |                                                                                              |
| `git push origin --delete <feature/feature-name>`  + enter                 | üìå Delete remote branch                                                                      |
| Pre-Pull: `git rev-list --count HEAD..@{push}` + enter                     | üîç Shows how many commits the remote has that you do not                                     |
|                                                                            |                                                                                              |
| **Step 6: Clean-up**                                                       |                                                                                              |
| ***6.1. Remote:***                                                         |                                                                                              |
| `git branch -r` + enter                                                    | üîç View all remote-tracking branches                                                         |
| `git branch -r --merged origin/main` + enter                               | üîç View remote branches that have been merged to main/master                                 |
| `git ls-remote --heads origin` + enter                                     | üîç View all references currently available in the remote repository                          |
|                                                                            |                                                                                              |
| `git push origin --delete <feature/feature-name>`  + enter                 | üìå Delete remote branch                                                                      |
| `git ls-remote --heads origin` + enter                                     | üîç Deleted remote branch no longer on the list                                               |
| `git branch -vv \| findstr ": gone]"` + enter                              | üîç View local branches with deleted upstream branches                                        |
|                                                                            |                                                                                              |
| ***6.2. Local:***                                                          |                                                                                              |
| `git branch` + enter                                                       | üîç View all local branches                                                                   |
| `git branch -d <feature/feature-name>` + enter                             | üöÆ Delete specified merged local branch                                                      |
| `git branch` + enter                                                       | üîç Verify deleted branch                                                                     |
|                                                                            |                                                                                              |
|                                                                            |                                                                                              |
|                                                                            |                                                                                              |
|                                                                            |                                                                                              |
| Pre-Pull: `git rev-list --count HEAD..@{push}` + enter                     | üîç Shows how many commits the remote has that you do not                                     |

#### One-Liner Delete
1. Delete local branches merged into current, excluding 'main' and current branch
```bash
`git branch --merged \| Where-Object { $_ -notmatch '^\*' -and $_ -notmatch 'main' } \| ForEach-Object { git branch -d $_.Trim() }`
```
2. Identify and delete merged remote-tracking branches (local refs only)
```bash 
`git branch -r --merged \| Select-String 'origin/' \| ForEach-Object { git branch -rd $_.ToString().Trim() }`
```
