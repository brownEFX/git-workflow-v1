# Practical Workflow Verification Table.
- Focuses on the **bare minimum** verification needed at each step.
- The `git status` command is your universal verification tool - it tells you everything you need to know about your current state in 90% of cases.
- This is a very clean and practical workflow verification table. Here's a refined version with some clarifications and optimizations:

## **Essential Verification Flow:**
1. **After clone:** `ls && git remote -v`
2. **After switch:** `git branch --show-current`
3. **After commit:** `git log --oneline -1`
4. **After push:** `git status -sb`
5. **After merge:** `git log --oneline -5 --graph`
6. **After cleanup:** `git branch -vv`


## **Critical Path - Minimal Verification:**
```bash
# 1. After any git operation
git status

# 2. After branching/committing
git log --oneline -3

# 3. After pushing
git status -sb  # Shows branch tracking status

# 4. After merging/cleanup
git branch -vv  # Shows tracking relationships
```

## **Git Workflow Verification Cheatsheet**

| **Stage**              | **Key Commands**                                             | **Purpose**            | **Verification**                             |
|------------------------|--------------------------------------------------------------|------------------------|----------------------------------------------|
| **1. Clone**           | `git clone <repo_url>`                                       | Create local clone     | `ls -la` (Unix) / `dir` (Windows)            |
|                        |                                                              |                        | `git remote -v`                              |
|                        |                                                              |                        |                                              |
| **2. Branch**          | `git switch main`<br>`git pull origin main`                  | Sync main              | `git status`<br>`git log --oneline -3`       |
|                        | `git switch -c feature/name`                                 | Create & switch branch | `git branch --show-current`                  |
|                        |                                                              |                        |                                              |
| **3. Work**            | Add/Edit files                                               | Make changes           | `git diff`                                   |
|                        | `git add .`                                                  | Stage changes          | `git status`                                 |
|                        | `git commit -m "msg"`                                        | Commit changes         | `git log --oneline -1`                       |
|                        |                                                              |                        |                                              |
| **4. Push**            | `git push -u origin HEAD`                                    | Push to remote         | `git status -sb`                             |
|                        |                                                              |                        | `git log --oneline origin/main..HEAD`        |
|                        |                                                              |                        |                                              |
| **5. GitHub PR**       | Create PR → Review → Merge                                   | Merge to main          | Verify CI passes                             |
|                        |                                                              |                        |                                              |
| **6. Post-Merge Sync** | `git switch main`                                            | Switch to main         | `git branch --show-current`                  |
|                        | `git fetch origin`                                           | Get latest             | `git diff origin/main`                       |
|                        | `git merge origin/main`                                      | Integrate changes      | `git status`                                 |
|                        |                                                              |                        | `git log --oneline -5 --graph`               |
|                        |                                                              |                        |                                              |
| **7. Cleanup**         | **Remote Branches:**<br>`git push origin -d branch1 branch2` | Delete remote branches | `git branch -r`                              |
|                        | **Local Branches:**<br>`git branch -d branch1 branch2`       | Delete local branches  | `git branch -vv`                             |
|                        | **Prune:**<br>`git fetch --prune`                            | Clean stale refs       | `git branch -vv`<br>`git remote show origin` |



## **Quick Verification Aliases (Add to ~/.gitconfig):**
```bash
[alias]
    verify = "!git status && echo '---' && git log --oneline -3"
    verify-push = "!git status -sb && echo '---' && git log --oneline origin/main..HEAD"
    verify-clean = "!git branch -vv | grep -E '\[gone\]|: gone'"
    verify-sync = "!git log --oneline --graph --decorate -10 --all"
```
