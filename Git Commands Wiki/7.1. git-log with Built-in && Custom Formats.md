# git Formats
## Git Built-in Format
| ***Format***  | ***Description***                            | ***Usage***                                                           |
|---------------|----------------------------------------------|-----------------------------------------------------------------------|
| `oneline`     | Displays abbr hash and commit subject line.  | `git log oneline`,`git show --oneline`, `git show --pretty=oneline`   |
| `short`       | Shows hash, author + subject line.           | `git log --pretty=short`, `git show --pretty=short`                   |
| `medium`      | Adds author date + full commit message body. | `git log --pretty=medium`                                             |
| `full`        | Replaces date with committer's name.         | `git log --pretty=full`, `git show --pretty=full`                     |
| `fuller`      | Incl author && committer details + dates.    | `git log --pretty=fuller`, `git show --pretty=fuller`                 |
| `raw`         | Shows commit exactly as it is stored         | `git log --pretty=raw`                                                |
| `reference`   | For citing them in other messages            | `git log --pretty=reference` (output: abbr-hash (subject, short-date) |

## Git Built-in Format
| ***Placeholder***   | ***Description***             | ***Usage***                                                         |
|---------------------|-------------------------------|---------------------------------------------------------------------|
| `%H`                | Commit hash                   | `git log --pretty=format:"%H"`                                      |
| `%h`                | Abbreviated commit hash       | `git log --pretty=format:"%h"`                                      |
| `%an`               | Author name                   | `git log --pretty=format:"%an"`                                     |
| `%ae`               | Author email                  | `git log --pretty=format:"%ae"`                                     |
| `%ad`               | Author date                   | `git log --pretty=format:"%ad"`                                     |
| `%ar`               | Relative Author date          | `git log --pretty=format:"%ar"`                                     |
| `%cn`               | Committer name                | `git log --pretty=format:"%cn"`                                     |
| `%ce`               | Committer email               | `git log --pretty=format:"%ce"`                                     |
| `%s`                | Subject (title) of the commit | `git log --pretty=format:"%s"`                                      |
| `%d`                | Ref names (branches, tags)    | `git log --pretty=format:"%d"`                                      |
| `%n`                | A newline character           | `git log --pretty=format:"%n"`                                      |
| `%C`                | Colour specification          | `git log --pretty=format:"%C(red)"`                                 |
| `%h - %an, %ar : %` | Full format                   | `git log --pretty=format:"%h - %an, %ar : %s" --since="2026-01-01"` |
| `%h - %an, %ar : %` | Full format                   | `git log --pretty=format:"%h - %an, %ar : %s" --since="2026-01-01"` |

- FULL Example:
    git log --since="6 hours ago" --pretty=format:"%C(yellow)Commit:%Creset %H (%h)%n%C(yellow)Tree:%Creset %T (%t)%n
    %C(yellow)Parents:%Creset %P (%p)%n%C(cyan)Author:%Creset %an <%ae> [%ad (%ar)]%n%
    C(cyan)Committer:%Creset %cn <%ce> [%cd (%cr)]%n%C(magenta)Refs:%Creset %d%n
    %C(green)GPG:%Creset %G? (%GS)%n%C(red)Subject:%Creset %s%n%C(white)Body:%Creset%n%b%n%(trailers)%n" 

- Use an Alias
  1. Run this once to set the alias:
     git config --global alias.lg "log --since='2 hours ago' --pretty=format:'%C(yellow)Commit:%Creset %H (%h)%n%C(yellow)Tree:%Creset %T (%t)%n%C(yellow)Parents:%Creset %P (%p)%n%C(cyan)Author:%Creset %an <%ae> [%ad (%ar)]%n%C(cyan)Committer:%Creset %cn <%ce> [%cd (%cr)]%n%C(magenta)Refs:%Creset %d%n%C(green)GPG:%Creset %G? (%GS)%n%C(red)Subject:%Creset %s%n%C(white)Body:%Creset%n%b%n%(trailers)%n'"

- Edit the .gitconfig File Directly
1. Open the file in your default editor:
   git config --global --edit
2. Find the [alias] section (or add it if it's missing).
3. Paste the following line under [alias]
   lg = log --since="2 hours ago" --pretty=format:"%C(yellow)Commit:%Creset %H (%h)%n%C(yellow)Tree:%Creset %T (%t)%n
   %C(yellow)Parents:%Creset %P (%p)%n%C(cyan)Author:%Creset %an <%ae> [%ad (%ar)]%n
   %C(cyan)Committer:%Creset %cn <%ce> [%cd (%cr)]%n%C(magenta)Refs:%Creset %d%n 
   %C(green)GPG:%Creset %G? (%GS)%n%C(red)Subject:%Creset %s%n%C(white)Body:%Creset%n%b%n%(trailers)%n"
