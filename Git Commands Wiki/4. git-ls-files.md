# git log
- View the commit history of a repository

| ***Stage***                                            | ***Key Commands***                                              |
|--------------------------------------------------------|-----------------------------------------------------------------| 
| `git ls-files` + enter                                 | View tracked files in repo                                      |
| `git ls-files -c` or `git ls-files --cached` + enter   | Lists all files currently in Git's index                        |
| `git ls-files -o` or `git ls-files --others` + enter   | Lists untracked files in the working directory.                 |
| `git ls-files -m` or `git ls-files --modified` + enter | Lists tracked files that have been modified but not yet staged. |
| `git ls-files -d` or `git ls-files --deleted` + enter  | Lists deleted from the working tree but are still in the index. |
| `git ls-files -o --exclude-standard` + enter           | Show untracked files while excluding ignored ones:              |
| `git ls-files -o -i --exclude-standard` + enter        | Check which files are currently being ignored                   |
| `git ls-files --unmerged` + enter                      | List files that are "unmerged" due to a merge conflict:         |
| `git ls-files --others --exclude-standard` + enter     |                                                                 |
