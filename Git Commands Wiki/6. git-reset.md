# git log
- View the commit history of a repository

| ***Stage***                                                 | ***Key Commands***                                     |
|-------------------------------------------------------------|--------------------------------------------------------| 
| `git reset --soft HEAD~1` + enter                           | Undo last local commit (keep changes in staging)       |
| `git reset HEAD~1` + enter                                  | Undo last local commit (unstage changes)               |
| `git reset HEAD~2` + enter                                  | Undo last 2 local commits (unstage changes)            |  
| `git reset HEAD~(n-1)` + enter                              | View changes between the nth and (n-1)th last commits. |
| `git rm --cached <file>` + enter                            | Unstage file from staging area and remove from index   |
| `git log --pretty=format:"%h - %an, %ar : %s" -n 5` + enter | View last n commits                                    |
| `git show --name-status + commitID` + enter                 | View files changed with status in specific commit      |
| `git show --name-status` + enter                            | View files changed with status in last commit          |
| `git reset --hard + commitID` + enter                       | Reset to specific commit                               |
