|---------------------------------------------------------|-----------------------------------------------------------------------------------------------|

| `git worktree add` + enter                              | ðŸ“Œ Create linked Working Tree                                                                 |
| `git worktree add <path>` + enter                       | ðŸ“Œ Creates a new branch named after the final folder in the path, based on the current HEAD.  |
