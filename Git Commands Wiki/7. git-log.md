# git log
- View the commit history of a repository

| ***Stage***                                | ***Key Commands***                                                   |
|--------------------------------------------|----------------------------------------------------------------------| 
| `git log -n 5 since='4 hours ago'` + enter | ðŸ“Œ View the commit history of a repository.                          |
| `git log <file-name>` + enter              | ðŸ“Œ View  only the history of a specific file.                        |
| `git log --oneline` + enter                | ðŸ“Œ View single-line commit with abbreviated hash and message.        |
| `git log -p` + enter                       | ðŸ“Œ View code changes introduced in each commit.                      |
| `git log --stat` + enter                   | ðŸ“Œ View summary of files changed + number of lines added or removed. |
| `git log --graph --oneline --all` + enter  | ðŸ“Œ View branch and merge history using an ASCII graph.               |
| `git log --grep="bug fix` + enter          | ðŸ“Œ Searches for specific keywords in commit messages.                |
|                                            |                                                                      | 

## Advanced git log
| ***Stage***                                                                | ***Key Commands***                                                             |
|----------------------------------------------------------------------------|--------------------------------------------------------------------------------| 
| `git log --grep=<pattern>`                                                 | Filter commit history by commit message using a specified pattern or regex.    |
| `git log --grep="Rename"` + enter                                          | Filter commit history by commit message pattern (case sensitive)               |
| `git log --grep=<pattern> --grep=<pattern> --grep=<pattern>`  + enter      | Filter commit history by commit message pattern-1 OR pattern-2 OR pattern-n.   |
| `git log --grep=<pattern> --grep=<pattern> --all-match`  + enter           | Filter commit history by commit message pattern-1 AND pattern-2 AND pattern-n. |
| `git log --grep="Rename" -i` or `... --ignore-case` + enter                | Case-insensitive match for the pattern.                                        |
| `git log --oneline --all --grep="Rename" -i` + enter                       | `--all` = Search history across all branches                                   |
| `git log --invert-grep=<pattern>`                                          | Filter commit history by only commits that do not match the pattern.           | 
| `git log` + enter                                                          | View commit history                                                            |
| `git log --oneline` + enter                                                | View concise commit history                                                    |
| `git log --oneline --decorate -5` + enter                                  | View last 5 commits in one line with decorations.                              |
| `git log --graph --oneline --decorate --all`                               | View commit history as a graph.                                                |
| `git log --stat`                                                           | View commit history with file changes summary.                                 |
| `git log --stat --since="2 days ago"`                                      | View commit history with file changes summary for the last 2 months.           |
| `git log -p <commitId>`                                                    | View changes made in a specific commit.                                        |
| `git log --author="<authorName>"`                                          | View commits made by a specific author.                                        |
| `git log --pretty=format:"%h - %an, %ar : %s"`                             | Custom format for viewing commit history.                                      |
| `git log --pretty=format:"%h - %an, %ar : %s" --since="2026-01-01"`        | Custom format for viewing commit history since a specific date.                |
| `git log --since="2 days ago"`                                             | View commits made in the last 2 days.                                          |
| `git log --since="2 weeks ago"`                                            | View commits made in the last 2 weeks.                                         |
| `git log --since="2023-01-01" --until="2023-02-01"`                        | View commits made within a specific date range.                                |
| `git diff [+ fileNm] + enter` + enter                                      | Compare file versions                                                          |
|                                                                            |                                                                                |
| `echo "" >> <file-name.md>` + enter                                        | Create at least 4 files                                                        |
| `git add <file-name>` + enter                                              | Add file to staging                                                            |
| `git commit -m "commit message"` + enter                                   | Commit file for next push. Repeat for remaining files.                         |
| `git log --pretty=format:"%h - %an, %ar : %s" --since="15 minutes ago"`    | View commit history in the last n minutes                                      |
| `git log --pretty=format:"%h - %an, %ar : %s" --since="15 minutes ago" -5` | View commit history in the last n minutes                                      |
| `git log --pretty=format:"%h - %an, %ar : %s" -n 5`                        | View last n commits                                                            |
| `git log --pretty=format:"%h - %an, %ar : %s" -5`                          | View last n commits                                                            |
