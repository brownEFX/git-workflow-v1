# git log
- View the commit history of a repository

| ***Stage***                                   | ***Key Commands***                                                                       |
|-----------------------------------------------|------------------------------------------------------------------------------------------|
| `git remote add <name> <url>` + enter         | Add and connect local git repo to new remote repository with the specified name and URL  |
| `git remote show origin` + enter              | Show detailed information about the remote repository named "origin"                     |
| `git remote -v` + enter                       | List all configured remote repositories and their URLs                                   |
| `git remote set-url origin <new_url>` + enter | Change the URL of the remote repository named "origin" to `<new_url>`                    |
| `git remote rename origin <new_name>` + enter | Rename the remote repository named "origin" to `<new_name>`                              | 
| `git remote set-head origin -a` + enter       | Set the default branch for the remote repository named "origin" to the current branch    |
| `git remote remove origin` + enter            | Remove the remote repository named "origin"                                              |
