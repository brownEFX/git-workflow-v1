| ***Commands**          | ***Description***                                                 |
|------------------------|-------------------------------------------------------------------|
| ls \| dir              | List dir contents                                                 |
| ls -FORCE              | List all files + subdirectories, including hidden ones            | 
| cd                     | Go to home directory.                                             |
| `.`                    | Represents the current directory.                                 |
| cd .                   | Refresh dir to see changes                                        |
| `..`                   | Represents the parent directory (one level up).                   |
| ../..                  | Go one level up, then go up another level                         |
| cd..                   | Navigate one level up                                             |
| cd ../sub-directory    | Go one level up, then cd into a sub-directory                     |
| cd ../..               | Navigate two levels up                                            |
| cd -                   | Toggle back to the previous directory you were in.                |
| cd \                   | Navigate to root dir [C:\]                                        |
| cd ~                   | Return to the current user's home directory  [C:\Users\user-name] |
| cd ~/Documents         | Return to the current user's home directory  [C:\Users\user-name] |
| pwd                    | Print Working Directory path                                      | 
| dir                    |                                                                   |
| ls                     |                                                                   |
| ls -Directory          | View normal directories                                           |
| ls -Force              |                                                                   |
| ls -Force -Directory   | View hidden and normal directories                                |
| mkdir                  | Create new working dir.                                           |
| rmdir                  | Remove an empty Directory                                         |
| rm file_name.extension | Remove file or directories                                        |
| diff                   | Difference between two files                                      |
| echo                   | Display a line of text                                            |
| \>                     | Redirect standard output to a file (overwrites the file           |
| \>>                    | Append standard output to the end of a file.                      |
