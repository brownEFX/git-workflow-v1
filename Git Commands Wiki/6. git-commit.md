# git log
- View the commit history of a repository

| ***Stage***                                            | ***Key Commands***                                                           |
|--------------------------------------------------------|------------------------------------------------------------------------------| 
| `git commit -m "commit message"` + enter               | Commit changes with message                                                  |
| `git commit -am "commit message"` + enter              | Stage and commit all changes with message                                    |
| `git commit`                                           | Commit changes with message in editor                                        |
| `git commit --amend` + enter                           | Amend last commit                                                            |
| `git commit --amend -m "new commit message"` + enter   | Amend last commit message                                                    |
| `git commit --amend --no-edit` + enter                 | Amend last commit without changing message                                   |
| `git commit --allow-empty -m "commit message"` + enter | Create empty commit with message                                             |
| `git commit --dry-run` + enter                         | Simulate commit without making changes                                       |
| `git commit -verbose` or `git commit -v` + enter       | Commit with verbose output. Opens editor with a diff of staged changes.      |
| `git commit -av`                                       | Automatically stages modified tracked files and shows the diff in the editor |
| `git commit --amend -v`                                | Opens the last commit's diff and message for editing                         | 
| `git commit -p`                                        | Commit changes interactively by selecting hunks.                             |
