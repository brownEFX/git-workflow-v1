# git log
- View the commit history of a repository

| ***Stage***                                                                  | ***Key Commands***                                                                         |
|------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| `git reflog` or `git log -g --abbrev-commit --pretty=oneline` + enter        | Recover lost commits or undo accidental history-rewriting ops (e.g. hard reset or rebase). | 
| `git reset --hard <commit-hash>` or `git reset --hard HEAD@{number}` + enter | Return to specific state using commit hash                                                 |
| `git branch <new-branch-name> <commit-hash>` + enter                         | Create a new branch at that specific commit                                                |
| `git cherry-pick <commit-hash>` + enter                                      | Apply the changes of a specific "lost" commit onto your current branch                     |
| `git reflog show` + enter                                                    | View the log for HEAD                                                                      |
| `git reflog --all` + enter                                                   | View reflogs for all references                                                            |
| `git reflog delete master@{2}` + enter                                       | Deletes a specific entry from the reflog (generally not used by end-users).                |
