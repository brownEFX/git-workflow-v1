# git log
- View the commit history of a repository

| ***Stage***                              | ***Key Commands***                                                                                  |
|------------------------------------------|-----------------------------------------------------------------------------------------------------|
| **Branch**                               |                                                                                                     |
| `git switch main` + enter                | Switch to main branch                                                                               |
| **Fetch**                                |                                                                                                     |
| `git fetch origin` + enter               | Get latest from remote without merging                                                              |
| `git fetch origin <branch-name>` + enter | Fetch a specific branch                                                                             |
| `git fetch --all` + enter                | Fetch from all configured remotes                                                                   |
| `git fetch --prune` + enter              | Fetch and remove ghost references                                                                   |
| **Merge**                                |                                                                                                     |
| `git merge main` + enter                 | Integrate fetched changes into local                                                                |
| `git merge --no-ff` + enter              | Forces the creation of a merge commit, preserves the branch history explicitly                      |
| `git merge --squash` + enter             | Combines all the changes from the source branch into a single commit on the target branch           |
| **Pull = Fetch + Merge**                 |                                                                                                     |
| `git pull` + enter                       | Fetch and Merge                                                                                     |
| `git pull origin main` + enter           | Pulls changes from a specific branch (e.g., main) on a specific remote (e.g., origin).              |
| `git pull --rebase` + enter              | Reapplies local commits on top of the new remote commits                                            |
| `git pull --ff-only` + enter             | Update branch if it can be done with a "fast-forward" (i.e., local hasn't diverged from the remote) |
