# git log
- View the commit history of a repository

| ***Stage***                                                            | ***Key Commands***                                                       |
|------------------------------------------------------------------------|--------------------------------------------------------------------------| 
| `git restore --staged <file-path>`                                     | Unstage specific files from Staging. Reverts index to match HEAD commit  |
| `git restore --staged .`                                               | Unstage all files in Staging. Reverts index to match HEAD commit         |
| `git restore --staged --source <commit-hash> <file-path>`              | Unstage specific files from the staging area to a specific commit state. | 
| `git restore --staged <fileNm>` + enter                                | Unstage a file.                                                          |
| `git restore <fileNm>` + enter                                         | Discard changes in working directory.                                    | 
| `git restore --source=<commitId> <fileNm>` + enter                     | Restore a file to a specific commit version.                             |
| `git restore --source=<commitId> .` + enter                            | Restore all files in working directory to a specific commit version.     | 
| `git restore --staged --source=<commitId> <fileNm>` + enter            | Unstage a file to a specific commit version.                             |
| `git restore --worktree --source=<commitId> <fileNm>` + enter          | Restore both WD and staging area to a specific commit version.           |
| `git restore --staged --worktree --source=<commitId> <fileNm>` + enter | Unstage and restore WD to a specific commit version.                     |
| `git rm --cached + <file-path>` + enter                                | Unstage file from staging area and remove from index                     |
