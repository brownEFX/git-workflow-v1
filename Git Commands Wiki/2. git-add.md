# git status
- Do your updates in a branch, not main

| ***Stage***                                                                               | ***Key Commands***                                                             |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------| 
| `git add + fileNm` + enter                                                                | ðŸ“Œ Add file to staging area                                                    |
| `git add .` + enter                                                                       | ðŸ“Œ Add all files in current directory to staging area                          |
| `git add -A` + enter                                                                      | ðŸ“Œ Add all files in Repository to staging area                                 |
| `git add -- . ':!file_to_exclude.txt'` + enter e.g. `git add -- . ':!.idea/' ':!file.md'` | ðŸ“Œ Stage all files except specific ones using pathspecs with negation patterns |
| `git add -u`                                                                              | ðŸ“Œ Add modified and deleted files to staging area                              |
| `git add --force <filename>`                                                              | ðŸ“Œ Force adding files that are normally ignored by .gitignore                  |
|                                                                                           |                                                                                |
| `git add -p` + enter                                                                      | ðŸ“Œ Interactively review and stage changes in chunks                            |
| `git add -p` prompt                                                                       | ðŸ“Œ Stage this hunk [y,n,q,a,d,j,k,s,e,?]?                                      |
| ***Key***                                                                                 | ***Action***                                                                   |
| `y` + enter                                                                               | Stage this hunk                                                                |
| `n` + enter                                                                               | Skip. Don't stage this hunk                                                    |
| `q` + enter                                                                               | Quit (stop reviewing hunks)                                                    |
| `a` + enter                                                                               | Stage this hunk and all later hunks in the file                                |
| `d` + enter                                                                               | Do not stage this hunk or any later hunks in the file                          |
| `s` + enter                                                                               | Split this hunk into smaller pieces                                            |
| `e` + enter                                                                               | Edit the hunk manually before staging                                          |
| `j` + enter                                                                               | Leave this hunk undecided, move to next                                        |
| `k` + enter                                                                               | Leave this hunk undecided, move to previous                                    |
| `?` + enter                                                                               | Show help                                                                      |
