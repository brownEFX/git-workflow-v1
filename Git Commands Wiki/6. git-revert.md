# git log
- View the commit history of a repository

| ***Stage***                          | ***Key Commands***                                                                          |
|--------------------------------------|---------------------------------------------------------------------------------------------| 
| `git revert HEAD` + enter            |                                                                                             |
| `git revert + commitID` + enter      | Revert specific commit                                                                      |
| `git revert <commit id>`             | Revert changes from a specific commit by creating a new commit.                             |
| `git revert --no-commit <commit id>` | Revert changes from a specific commit without creating a new commit immediately.            |
| `git revert -n <commit id>`          | Revert changes from a specific commit without committing the changes (same as --no-commit). |
| `git revert --continue`              | Continue the revert process after resolving conflicts.                                      |
| `git revert --abort`                 | Abort the revert process and return to the state before the revert began.                   |
| `git revert HEAD ~n`                 | Revert the commit that is n commits before the HEAD.                                        |
| `git revert HEAD ~2..HEAD`           | Revert a range of commits from HEAD~2 to HEAD.                                              |
