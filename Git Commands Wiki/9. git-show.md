# git log
- View the commit history of a repository

| ***Stage***                                    | ***Key Commands***                                |
|------------------------------------------------|---------------------------------------------------| 
| `git show + commitID` + enter                  | View specific commit details                      |
| `git show` + enter                             | View last commit details                          |
| `git show --stat` + enter                      | View last commit summary                          |
| `git show --name-only + commitID` + enter      | View files changed in specific commit             |
| `git show --name-only` + enter                 | View files changed in last commit                 |
| `git show --name-status + commitID` + enter    | View files changed with status in specific commit |
| `git show --name-status` + enter               | View files changed with status in last commit     |
| `git show --pretty=oneline + commitID` + enter | View specific commit in one line                  |
