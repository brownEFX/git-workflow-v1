# git log
- View the commit history of a repository

| ***Stage***                                          | ***Key Commands***                                                                                           |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| `git stash` or `git stash push` + enter              | ğŸ“Œ Save changes: Saves uncommitted local changes and reverts the WD to the HEAD commit.                      |
| `git stash push -u` OR `--include-untracked` + enter | ğŸ“Œ Save changes: Saves changes including files not yet tracked by Git.                                       |
| `git stash push -u -m "message"` + enter             | ğŸ“Œ Save changes: Saves only tracked and untracked files (excluding ignored)  in WD with a descriptive label. |
| `git stash push -a` + enter                          | ğŸ“Œ Save changes: Saves everything, includes both untracked and ignored files.                                |
| `git stash push -m "message"` + enter                | ğŸ“Œ Save changes: Saves your work with a descriptive label.                                                   |
| `git stash push -a -m "message"` + enter             | ğŸ“Œ Save changes: Saves all changes in WD with a descriptive label.                                           |
| `git stash push -p` + enter                          | ğŸ“Œ Save changes: Saves partial stash, interactively select specific "hunks" of code to stash.                |
| `git stash branch <branch-name>` + enter             | ğŸ“Œ Creates a new branch and applies the stashed changes to it.                                               |
| `git stash list` + enter                             | ğŸ“Œ List stashes: Shows all saved stashed changes, identified by an index like stash@{0}.                     |
| `git stash pop` + enter                              | ğŸ“Œ Reapply changes: Applies the most recent stash and removes it from the stack.                             |
| `git stash pop '<stash@{n}>'` + enter                | ğŸ“Œ Applies stash at index n and removes it from the stack.                                                   | 
| `git stash apply` + enter                            | ğŸ“Œ Reapply changes: Applies latest changes but keeps the stash in the stack for later use.                   |
| `git stash apply '<stash@{n}>'` + enter              | ğŸ“Œ Restores Specific: Restores stash at index n but keeps it in the stack.                                   |
| `git stash apply --index` + enter                    | ğŸ“Œ Restores stashed changes to their original staged state                                                   |
| `git stash show -p` + enter                          | ğŸ“Œ Full View: Displays the actual line-by-line code diff of the most recent stash.                           |
| `git stash show 'stash@{n}'` + enter                 | ğŸ“Œ View contents - Summary View: Displays a summary of files changed within a specific stash index.          |
| `git stash drop "stash@{n}" or n` + enter            | ğŸ“Œ Remove stashes - Delete One: Permanently deletes a specific stash from the stack.                         |
| `git stash clear` + enter                            | ğŸ“Œ Remove stashes - Delete All: Permanently deletes every stash in the repository.                           |
