# git log
- View the commit history of a repository

| ***Stage***                                 | ***Key Commands***                                 |
|---------------------------------------------|----------------------------------------------------| 
| `git ls-tree -r HEAD --name-only` + enter   | View all files tracked in a remote-tracking branch |
| `git ls-tree -r HEAD --name-status` + enter | View all files tracked in a remote-tracking branch |
| `git ls-tree -r HEAD --name-only` + enter   | List local files                                   |
| `git ls-tree -r origin/main` + enter        | List and include details (mode/SHA)                |
| `git ls-files` + enter                      | List all tracked files in the index                |
