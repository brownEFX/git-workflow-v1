# git log
- Syntax: `git ls-tree [<options>] <tree-ish> [<path>...]`
- View the commit history of a repository

| ***Stage***                                             | ***Key Commands***                                                                         |
|---------------------------------------------------------|--------------------------------------------------------------------------------------------| 
| **Local**                                               |                                                                                            |
| `git ls-files` + enter                                  | List all tracked files in the index                                                        |
| `git ls-tree <commit_hash>` + enter                     | View contents of specific commit                                                           |
| `git ls-tree HEAD --name-only` + enter                  | View contents of the current branch's latest commit                                        |
| `git ls-tree <branch-name>` + enter                     | View contents of specific branch                                                           |
| `git ls-tree -r HEAD --name-only` + enter               | View all files tracked in a remote-tracking branch                                         |
|                                                         |                                                                                            |
| **Remote**                                              |                                                                                            |
| `git ls-tree -r origin/main --name-only` + enter        | View files currently tracked in the origin/main branch List and include details (mode/SHA) |
|                                                         |                                                                                            |
| **Common Options**                                      |                                                                                            |
| `git ls-tree -r` + enter                                | Recursively lists the contents of subdirectories within the specified tree.                |
| `git ls-tree -l` + enter                                | Long format: Includes the object size in the output (only for blobs/files).                |
| `git ls-tree -t` + enter                                | Shows the tree itself and the entries in it.                                               |
| `git ls-tree -z` + enter                                | Uses a NUL character to terminate lines instead of a newline, useful for scripting         |
|                                                         |                                                                                            |
| **Combos**                                              |                                                                                            |
| `git ls-tree <branch-name> -r -l -t` + enter            | Recursively lists the contents of subdirectories within the specified tree.                |
| `git ls-tree <branch-name> --name-only -r -t` + enter   | Recursively lists the contents of subdirectories within the specified tree.                |
| `git ls-tree <branch-name> --name-status -r -t` + enter | Recursively lists the contents of subdirectories within the specified tree.                |
